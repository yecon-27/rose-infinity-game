# 《Rose Infinity · 玫瑰无限》剧情大纲 · 分幕剧本

> 这是**剧情的源头文件**:美术清单(`docs/art-asset-plan.md`)喂美术、`lib/story.ts` 喂代码,都从这里派生。
> 写作原则、语气、禁词见 `docs/writing/dialogue-style-guide.md`。
>
> **一句话**:两个学生情侣,回避×焦虑、一体两面,明明相爱,却在一连串"看起来都对"的选择里悄悄走散。玩家扮演 Vera:先活一遍 → 回到记忆里"看见"当年没看见的伸手 → 学着接住。结局是治愈系(《花束般的恋爱》式的成长,不必复合)。
>
> **铁律**:两个角色**同等可理解、同等有责任、同等值得心疼——没有坏人**。素材来自真实经历,但一律典型化,不做私人日记、不渲染委屈、不替谁讨说法。

---

## 弧线总览

**甜蜜期 → 爆发期 → 僵持期 → 分手 → 半年后 → 治愈**;"回看/看见"贯穿事后。
温度从热滑到冷,却找不出是哪一天变的。反复出现的母题:**"未来的小房子"**(两个学生却把退休、养什么狗都规划好了——憧憬越具体,坠落越疼)。

---

## 甜蜜期

### 幕1 · 黑客松那夜 ✅（已实装 lib/story.ts）
- **场景**:深夜黑客松,会场 → 楼梯间。
- **剧情**:他埋头赶代码,一次次"等我搞完";她一次次伸手（捶肩/握手/要眼神）都没被接住;她记得他吃清淡,点了荷叶包鸡;楼梯上他倒完苦水,她先肯定他、再用事实说出自己的委屈——他接住、道谢。**一次成功的修复,是全片"做对的一次"的范本。**
- **Vera 的面/问题**:把自己的饿和需求排在后勤与肯定他之后,"不吵不闹"——优雅,也是自我消音的开始。
- **Sean 的面/问题**:一过载就对她下线,"等我搞完"往后拖;微观清醒、宏观瞎。
- **种子**:他的"等我搞完";她的"把自己咽下去"。

### 幕2 · 挑衣服
- **场景**:商场服装店 / 试衣间外。
- **剧情**:出门前 Vera 用自己的审美把 Sean 从头到脚捯饬一遍。
- **Vera 的面/问题**:"就这件,不许换"——此刻是可爱的强势;同一只手日后会变成控制。
- **Sean 的面/问题**:被摆弄的宠溺无奈,享受被照顾——把这份照顾当成理所当然的苗头。
- **种子**:照顾里的掌控欲(=日后查岗的同一只手)。

### 幕3 · 分享《非暴力沟通》
- **场景**:校园长椅 / 宿舍楼下 / 咖啡馆;手里一本《非暴力沟通》。
- **剧情**:Vera 坦白自己性子急,这阵子总冲他发脾气,觉得该练练非暴力沟通;Sean 说很欣赏她这股肯改的劲,"其实你也一直在包容我",让她学到什么多跟他分享。两人真诚、互相欣赏、想一起成长。
- **Vera 的面/问题**:主动承认急脾气、想改——上进而可爱。但这脾气正是日后**查手机爆发**要喷的那张焦虑的脸;此刻她知道、在努力。
- **Sean 的面/问题**:温柔、愿意一起长进;可他把她的"包容"当成值得感激的美德,没看出那也是她在**硬撑、在把自己收起来**。
- **反讽/伏笔（本幕的魂）**:他们此刻**正握着沟通的工具**、还在互相欣赏地讨论怎么更好地爱——可日后 drift 里没能用上。和黑客松的修复一样:**他们做得到,只是撑不住。**
- **关键台词方向**:
  - Vera:"我性子太急了,这阵子老冲你发火……我想学学非暴力沟通。"
  - Sean:"我挺喜欢你这股劲的。其实,你也一直在包容我。你要是学到什么,多跟我讲讲。"

### （可选）未来的小房子 · 憧憬
- 母题镜头:两个学生把退休、养什么狗、房子朝向规划得钉是钉。可并入幕3 或单作一小拍。憧憬越具体,后面越疼。

## 爆发期

### 幕4 · 查手机
- **场景**：某人的宿舍房间 / 约会散场后的角落，深夜、紧绷。
- **剧情**：不安全感到顶点，Vera 翻了他的手机、关注/点赞记录，看到很多异性、一些"不对"的痕迹（多半不是真出轨）。越查越信，质问、爆发。这是整段关系里唯一"大声"的一段。事后两人都羞耻，缩回冷处理——僵持期由此开始。
- **Vera 的面/问题**：幕3 里她想改的急脾气，在这里喷了出来（焦虑那面失控）；她把不安变成质问、翻查，越界。
- **Sean 的面/问题**：被不信任刺伤后关上、下线，用冷回应；也可能确有让她不安的疏忽（回复慢、社交含糊），却不肯正面安抚。
- **反讽**：她幕3 才说要练非暴力沟通——工具在手，压力一来还是没接住。
- 📌 **吵架无法逐字写死 → 最适合接 LLM 智能回应。**

## 僵持期

### 幕5 · 发烧夜（扳机 · 两人都有错）
- **场景**：分屏 / 两地——Sean 发烧缩在宿舍床上，手机亮在手边；Vera 在自习室对着老师交代的任务，手机扣在桌上。
- **剧情**：Sean 发烧想让 Vera 来陪；Vera 被任务绊住，责任感强放不下，何况来了也帮不上什么忙。Sean：“以后结婚了，工作也比家庭重要吗？”谁都有道理，谁也没接住谁。
- **Vera 的面/问题**：用"责任"当挡箭牌，把体贴收在一个能自我说服的安全距离里；她的回避戴着"懂事、负责"的面具，没去找中间地带（一个电话、一句软话、晚点去看一眼）。
- **Sean 的面/问题**：把一次没来上升成对整段未来的审判，用愧疚绑架，要"被排在第一位"；他的焦虑装成一个价值观质问。
- **本幕的意义**："没有坏人"的一块基石。也是那个"小得可笑的扳机"——她那晚可能就已经决定了，只是拖了很久才说。
- ⚠️ 写作提醒：别写成"工作 vs 感情"谁对谁错的辩论；两个立场都成立、都有盲点，让玩家自己掂量。

## 结束 & 事后

### 幕6 · 分手夜（相爱却走不下去）
- **场景**：收走留在对方宿舍的东西 / 宿舍楼下告别。一个轻得不像样的袋子、门口的灯。
- **剧情**：**不是无声风化。** 两个**深爱**的人，有挽留、有眼泪。可两人心里都清楚同一件事——**在一起还是会一直吵**（查手机、发烧夜那些没解开的结还在），**校园恋情太难有结果**（快毕业、要异地、都还太年轻、各有各要奔的路）。于是含着泪，做了一个清醒的决定。**爱是真的，分开也是真的。**
- **挽留的戏（重要）**：至少一次真正的"别走 / 我们再试试"——玩家(Vera)可以挽留，Sean 也可能挽留。但那句"可我们在一起还是会吵个不停"横在中间，谁也绕不过。不是谁不爱，是两个相爱的人一起承认了现实。
- **两人的面**：都疼、都不舍、都认。这里**不是**冷漠的下线，是"相爱却走不下去"的痛。
- **延迟的痛（接幕7）**：分开之后 Sean 反而异常平静、甚至睡得好——那是他回避式的麻木；真正的痛半年后才砸下来（幕7）。Vera 走出来得慢一点。
- ⚠️ 写作提醒：别滑成狗血哭喊，也别滑回冷漠风化；是"哭着把话说明白、然后好好放手"。

### 幕7 · 半年后
- **场景**：还是那家便利店，她一个人。
- **剧情**：一句没经过任何修饰的真话（"要，谢谢，今天有点冷"）——她说得慢，但说完整了。她开始能好好说话了。痛过去了，人长出来了一点。

## 治愈 / 尾声

- 不复合（参考《花束般的恋爱》）：两人后来各自明白那段是真的爱过，也一直相信有真的爱情；各自带着复盘，长出一点爱人的能力。玫瑰不是为这一对开的，是为他们各自往后的人生开的。
- 视觉母题：`rose-bud`（花苞）→ `rose-bloom`（盛放）。

---

## 回看 / 看见（贯穿事后，玩法灵魂）

活过一遍后，回到记忆里，找出当年**没看见的伸手**，点开浮现对方那一侧当时没说出口的心情，再学着"接住"。
- **黑客松回看**（已实装 `/look`）：看见 Sean 那一侧（他不是没看见你，是先被自己淹没了）；"接住"改成互相/对自己——"这次先开口的是你：我也累了，你呢？"（不是再哄他一次）。
- **真正的"接住"（回到那一刻补救）最适合放在她当年确实没接住的幕**：查手机后的冷、发烧夜、分手夜。黑客松那幕她本就做对了，回看的 payoff 是共情 + 自我看见。

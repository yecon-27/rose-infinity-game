# 《Rose Infinity · 玫瑰无限》设计基准 & 台词风格圣经

> 本文档分两部分:
> **第一部分 · 设计基准**——游戏是什么、要给玩家什么体验、机制怎么运转。所有场景、选项、旁白、结局都以此为准绳。

> ⚠️ 本作已彻底废弃早期的"过滤器 / The Filter"设定。凡与下述基准冲突的旧文本,以本基准为准。

# 第一部分 · 设计基准

## 0.1 一句话

Rose Infinity(玫瑰无限):一个关于"回避的人也能学会好好爱"的治愈系叙事游戏。
你会先亲历一段温暖的关系,如何在一连串"看起来都对"的选择里悄悄流走;然后,你会有机会学着做不一样的选择——玫瑰不是给谁的特权,是每个愿意学的人都能拥有的。

## 0.2 名字的意思(整个游戏的脊椎)

- **玫瑰** = 爱情、幸福,人人都渴望、也人人都能拥有的东西。
- **无限** = 它永远在那儿。不会因为你搞砸过一次就消失;爱是可以重新学、重新来的。
- **但有前提**:得先看懂自己是怎么把它推开的,再学着不推。幸福不是天赋,是练出来的。
- 所以**结局的终点是治愈,不是悲剧**。悲剧只是"如果不改变"的那条路——让你亲身走一遍,好知道要改的是什么。

### 登场人物

- **Vera(玩家 · 她)**——主视角(第一周目全程)。用照顾表达爱、能把真话平静说出口的人。底下也焦虑:那份照顾(挑衣服、盯补给、偷买鱼油)和后来的查手机是同一只手——爱的那张脸,和抓不住时的那张脸。名字取拉丁语"真实",扣住全片命门:说不说得出真的那句话。
- **Sean(NPC · 他)**——广东人,写代码。他爱她,爱的方式是"把她的事当自己的事"和记得。但一过载就对她**下线**:用圆滑和"等我搞完"往后拖。他微观清醒(知道自己在躲)、宏观瞎(没算到一句句加起来的后果)。二周目转到他的视角,揭示他不是不爱。
- 旧名 阿沉 / 阿默 作废,代码与文案中一并替换。

## 0.3 角色真相(推翻旧"过滤器"框架的根本)

1. **他不是病人,是个大部分时候都很好的人。** 温暖是主旋律(陪她点外卖、吐槽同事、记得她不吃香菜)。那种僵冷的短对话**不是常态**,是关系已经开始僵了以后才有的。
2. **一体两面:回避的人,底下也是焦虑的。** 热恋期问题就已经埋好了——焦虑那面会露成占有欲、控制欲、或对对方的隐性消耗,藏在蛛丝马迹里。当时看着全像"太爱你了",事后才认出:那是同一个东西的另一张脸。
3. **僵持期,他知道自己在躲,他只是算过了成本。** 他清楚她想要什么,只是这会儿去关心成本太高(累、麻烦、要专门跑一趟),于是给一句圆滑的、听着体贴、其实什么都不用付出的话("有事叫我""多喝水"),把开口的难题又推回给她。他**微观上清醒**(知道这一句在躲),**宏观上瞎**(没算到这些一句句加起来,正让她慢慢放弃)。
4. **僵持期,她内心翻江倒海,却怕显得太感性、太"离不开你"。** 所以她体面地收住,回一句得体的。她不是不难受,是不肯让难受被看见——一被看见,就掉价了。
5. **玩家体验的落点**:不是"我在演一个冷漠的人",而是"我每一步都选得挺正常的,怎么就走到这了"——然后,最后一层:"原来我可以不这样。"

### 一体两面怎么驱动(废弃"情绪天平"数值,改情境触发)

早期代码用一个 `balance: -100~+100` 的数值刻度决定此刻露哪张脸——**这是要废掉的"情绪天平"的残影**,别用仪表盘表现人心。改成:

> **此刻露哪张脸 = 阶段(phase) × 这一拍发生了什么(情境触发)。**

- **情境触发**(由场景作者按拍标注,离散、不累积):
  - `approached` 被逼近——对方在靠近"我们"、索要亲密或承诺 → **回避那面**(往后撤 / 圆滑 / 岔开)
  - `neglected` 被冷落——感到被忽视、被推开、不安全 → **焦虑那面**(查、控制、冷嘲、试探)
  - `secure` 安稳——没越界、没威胁 → **正常那面**(甜蜜期=活的;僵持期=客气的稳)
- **阶段**决定语气温度(warm 热 / strained 冷),见 0.5。
- 情绪**不累积在一个隐藏刻度上**。回避和焦虑都是"此刻这拍戳到了什么"触发的,过去就过去。这既不病理化,又让每一次露脸都对应到一个具体的、玩家事后"看得见"的瞬间。

> 给 prompt / 引擎:把 `balance` 换成 `trigger: "secure" | "approached" | "neglected"`(或等价物),由场景按拍给。`partnerTone`(对方这句怎么落地)可保留——它是对当前这句的分类,不是累积刻度。

## 0.4 核心玩法:看见,不是选择

**核心动词是"看见 / 找出",不是"挑选项"。** 这个游戏的主题就是"学会认出对方在伸手",所以把这件事本身做成玩法。

**框架 / 开场钩子**:关系已经结束了,你说不清是哪天散的。你回到记忆里,一段段重新看,**找出当年没看见的那些"她在伸手"的瞬间**,学着把遗憾接回来。人都有遗憾;学会挽回遗憾,**有的故事就还有结局**。

**两段结构**:
1. **前段 · 先活一遍(约 5–7 分钟)**——温暖、跟手,你做着一个个自然的选择,然后她就那么淡出了。这一段不为改变结局,只为**先让你疼一下、先爱上她**。
2. **主体 · 回到记忆里,重新看**——玩法的重心在这。在温暖回忆的表面底下,**主动去发现那些藏起来的伸手**(她划走香菜时的半秒停顿、"随便你点吧"背后的僵、黑客松那只被抽回去的手),看到她/他那一侧当时真正的心情,学着重新接住。**你看懂得越真,玫瑰开得越近。**

**深度来自**:发现(不是被动挑选)、顿悟("啊……原来那时候就开始了")、以及"看懂"真的会解锁东西——不是白选。**看见 = 玩法 = 主题 = 治愈的钥匙,三合一。**

**仍然成立的几条**:
- 没有自由输入、没有任何"改写装置"(彻底告别过滤器)。
- 那些开口的瞬间,回避选项永远伪装成最成熟、最体贴、最不小题大做的那个——所以第一遍你自然会选它,回看才认出那是在推开对方。
- 当下不显示"心里想的 vs 说出口的"落差;落差全部延后到"看见"那一刻才揭开。
- 对方(Sean / Vera,AI 现场生成回应)也在递台阶(发烧也说"没事你忙")。两个人互相留出口,谁都没做错**单独任何一件事**——这是真实,不是谁的病。

**天花板(诚实记一笔)**:它是一部**需要用心去看的互动短篇**(Florence 那一挂,约 10–15 分钟),不是操作爽游。抓力来自开场钩子、发现时的顿悟、和 AI 让人物是活的。

**待定(下次要钉死)**:"回去找出那一次伸手 → 接住"这个循环,玩家手指具体怎么操作(倒带 / 放慢 / 盯住那一帧 / 一次要付代价的接住?)——方向已定,手感细节还没定,先记在这。

## 0.5 情绪弧线(关系的一生)

真实的关系不是一开始就冷的。它有四个阶段,戏就在于**从热滑到冷,却找不出是哪一天变的**:

1. **甜蜜期**——大量真实的甜,让玩家真的投入、存下感情。但**问题这时就已经埋好了**:焦虑那面偶尔露头——查岗、"你怎么又跟他们出去"、把对方的时间悄悄占满。当时都像浓烈的爱,只是蛛丝马迹。
2. **爆发期**——藏着的焦虑总有一次会喷出来。引子是**没有安全感**:**她**查他的手机、翻社交软件的关注/点赞记录,看到很多异性、或一些"不对"的痕迹。**多半不是真的出轨**,是不安全感在含糊里找到了燃料——越查越信,最后质问、爆发。这是整段关系里唯一"大声"的一段。**两个人都有点回避,只是表达不同**:她的回避会先翻成焦虑(查、控制、质问),他的回避是往后缩(冷、圆滑、算成本)。所以爆发过后不是继续吵,而是**两个人一起缩回冷处理**,谁也不肯再碰那个话题——僵持期由此开始。
3. **僵持期**——爆发过后,两个人都退回冷处理。这里才是那种圆滑躲闪(他)+ 体面收住(她)的短对话;"发烧夜"那类小得可笑的扳机就发生在这一段。她那晚就决定了,只是拖了很久才说出口。
4. **风干 / 分手**——没有导火索,一句"最近都挺忙的"就结束了。他甚至异常平静,当晚睡得很好。

**事后**——半年后,痛才砸下来,你第一次看懂了前面每一步(包括甜蜜期那些蛛丝马迹,当时到底是什么)。
**治愈回合**——带着看懂的东西,学着重来一次。这一次,玫瑰开了。

### 两种语气(非常重要,别再把整段关系写成冷的)

- **热的语气**(甜蜜期为主):活的、具体、有来有回、会闹会贫,句子该长就长。
- **冷的语气**(僵持期以后):才是"话极短、圆滑躲闪、体面收住"那一套。
- 本圣经第二部分"话极短、冷处理"的规则**只适用于僵持期及以后**,不要拿去写甜蜜期。前期若也这么写,整段关系读着就像两个早已冷掉的人——这正是"不像真实发生"的根源。

### 每个阶段:自己的温度,也有自己的玩法(治单调的总纲)

单调的病根 = 四个阶段用同一种温度 + 同一个套路(检视→小活动→开口)。解法是让阶段本身成为变化:

| 阶段 | 温度 | 玩法的感觉 |
|---|---|---|
| 甜蜜期 | 热、活、闹 | 轻快、表达型:一起做事、一起憧憬未来、发现"只有我俩懂"的默契。这里**没有**"从合理选项里挑"的节拍——这里是让玩家爱上她 |
| 爆发期 | 烫、大声 | 高压、侵入式:亲手划她的手机、一条条看关注。热度和交互都和别处截然不同 |
| 僵持期 | 冷 | 慢、克制:才是"挑一句体面话"的低温节拍(现有笔法留给这里) |
| 事后 / 治愈 | 静 | 回看、认出那些当年没看见的伸手 |

温度有高低,摔下来才疼;玩法一阶段一个样,才不会玩到一半就预判。

### 甜蜜期的质地(这一对的"那个东西")

参考《花束般的恋爱》:极具体的共同默契 + 是活的 + 甜里已埋结局的种子。这一对共享的具体是:

- **对未来无比具体的憧憬**(核心):一个未来的小房子、什么时候退休、养什么品种的狗。明明只是两个学生,却把未来规划得钉是钉铆是铆。
- **事业上互相当自己的事**:他忙黑客松,她带他出去放松;她项目卡住,他拿过来当自己的做。
- **她用照顾表达爱**:带他吃好店、用自己的审美给他挑一身衣服、盯他晚上吃没吃补给、偷偷攒钱给他买鱼油。
- **智识上的亲密**:一起聊哲学,交换"同一件事男生女生怎么看"。

**埋在甜里的种子(一体两面,最关键)**:
她这些照顾——挑衣服、盯补给、偷买鱼油——和后来的查手机,**是同一种爱、同一种想抓住,只是甜的那张脸**。甜蜜期它叫"为你好、这么黏人真可爱";爆发期同一个东西露出另一张脸,就成了控制、查岗。**别把照顾写成纯甜,要甜得稍微有点密不透风**——事后回看才认得出,原来那时就开始了。
另一颗种子:两个学生把未来规划得那么满,可这份连接建立在"共同憧憬"上,还没被现实(毕业、事业、异地、那个深圳 offer)称过重量。憧憬越具体,后面被现实压垮时越疼。

## 0.6 结局观

**总原则**:结局不在第一遍分叉。第一遍注定走到分手——那是"教材",让你亲身疼一次;真正的分叉在**看懂之后的"重来一次"**。这样才对得起"先失去、看懂了,才配得上幸福"。

四种收束:

1. **风化(糊涂的结束)**——第一遍默认落点。关系没有死因,谁也说不清哪天散的。不是终点,是把你推进"事后"的疼与看懂的起点。
2. **清醒的告别**——你看懂了,但太晚,或看懂了却没握住。两个人第一次把"我们好像都在等对方"说出口,然后好好分开。成长了,这段却没复合。不是失败,是干净。
3. **玫瑰开(治愈)**——走过"看懂"之后的重来一遍,你认出了那些伪装成没事的伸手、并付出代价接住,爱才留住。**但"玫瑰开"不等于这一对复合**(见下)。
4. **"无限"框**——任何没开花的收尾,最后都留一句"玫瑰还在,下次"。失败不是死路,是循环,是爱可以重新学。这就是标题。

**"玫瑰开"的确切含义(参考《花束般的恋爱》)**:
不是这一对破镜重圆。是两个人后来各自明白——**那段是真的爱过,不是错觉;他们也一直相信世上有真的爱情,想握住真的幸福**。后来各自遇到合适的缘分,带着对上一段的复盘,长出了一点点爱人的能力。玫瑰不是为这一对开的,是**为他们各自往后的人生**开的。"大家都能拥有爱情和玫瑰"——前提是先在一段里输过、并且看懂过。

**判定"学到了"(倾向方案,细节待定)**:学到 = 你现在**认得出**那个伪装成"没事"的伸手瞬间,并且愿意付出"违背省力直觉"的代价去接住。后台静默计数,不显示刻度;反馈全在戏里(她的眼神、玫瑰的意象),不打分。

## 0.7 词汇红线(玩家可见处一律不出现)

| 废弃(病理化 / 机械感) | 换成 |
|---|---|
| 过滤器 / The Filter / 改写 | 整个概念删除 |
| 情绪天平 / 刻度盘 / 数值 | 删。不用仪表盘表现人心 |
| 过滤强度 / 穿透 / 裂纹 | 删 |
| "回避型依恋"作为标签 | 只活在设计文档里,不进玩家眼睛。玩家看到的是一个具体的人,不是一个诊断 |

## 0.8 叙事红线:个人经历是材料,不是伤口展览

本作改编自真实经历——细节编不出来,这是最大优势;但最大的风险是**把游戏写成给自己的日记,而不是给玩家的体验**。全程用三条检查把关:

1. **删减测试**——凡是某段"舍不得删、但删掉故事不受影响"的,大概率只对作者有意义,不是玩家需要的。私人的意义不进游戏。
2. **角色厚度**(尤其防把 Vera 写成完美受害者)——素材来自单侧视角,最容易把"发出需求—然后失望"的一方写薄成功能性角色。**Vera 必须有自己的生活、自己的犹豫、以及一两处她也做得不完美的时刻**(她的照顾里有控制,她的"不吵不闹"里有记分和自我牺牲的姿态)。她越立体,越像作品而不是控诉。Sean 同理要厚(见二周目他的视角)。
3. **基调检查**——不控诉、不自怜、不替谁讨说法。主角每一步都合理,她还是走了——让玩家自己走进那个位置、自己意识到发生了什么。凡有"解释 / 辩护 / 渲染委屈"味道的段落,一律改掉或删掉。

**一句话**:玩家的疼要靠他自己的选择一层层叠出来,不是靠我们哭给他看。

# 第二部分 · 台词风格圣经

> 依据:回避型依恋社区真实语料(知乎/虎扑相关讨论)+ 张爱玲对话技法(《倾城之恋》《半生缘》《爱》)。
> 所有台词(阿默的话、活动结果、旁白)写作前先过这份检查表。
> 注:第五节"真情侣感"是本作的定海神针,和上面的设计基准完全一致——请优先内化。

---

## 一、真实回避型的说话方式(社区语料归纳)

**高频原话**(可直接入库使用):
- 「嗯」「好」「行」「都可以」「你定」
- 「你先忙」「没事」「还好」「睡了」
- 「我需要安静一下」「先这样吧」「我没做好准备」
- 「到了说一声」「吃了吗」「路上小心」

**核心规律**:
1. **话极短**。冷淡期一个字,常态一句话。超过 15 字的台词必须是事务性内容(地址、安排、预算)。
2. **在乎全部位移成事务/行动**,绝不命名情感:
   - 「多买了,帮我吃」= 我特意给你买的
   - 「到了说一声」= 我会一直担心到你到家
   - 把糊的那串抽走自己吃 = 心疼你
3. **冷处理,不表演**:不回消息、转移话题、突然聊工作。被逼近时说「先这样吧」,不吵架、不解释。
4. **接受"无需求感"的话**(「嗯,好的,你先忙」),抗拒撒娇和高浓度表达。
5. **焦虑闪现时是冷嘲,不是吵闹**:「哦,现在知道找我了」「行,你忙」——音量不变,温度骤降。

## 二、张爱玲对话技法

1. **平静的陈述句砸人**:「世钧,我们回不去了。」——最重的话用最平的语调,没有感叹号。
2. **情绪位移到物件/天气**:不写"她难过",写她把小票折成方块、把竹签插得整整齐齐。
3. **表面全是废话,底下全是暗流**:「哦,你也在这里吗?」
4. **反高潮**:关键时刻说的永远是最日常的话。
5. **沉默是台词**:很多"回应"应该写成旁白里的动作,而不是让她开口。

## 三、禁词表(出现即重写)

> ⚠️ **语气分级**:下面这份是**僵持期(冷语气)**的默认禁令。**甜蜜期(warm)要放宽**——允许长句、允许会闹会贫、允许感叹号、允许两人**共享的旧梗**、允许有生气。别把冷语气的克制套到甜蜜期,否则整段关系读着又像两个早就凉了的人。
> 但以下三条**两个阶段都禁**:①撒娇/嗲、②让角色"知道自己在可爱"的台词、③网文梗(殉职/上道/拿捏/破防)。核心不变:**幽默是熟的,不是甜的**。

- 撒娇/贫嘴:傻子、~、!、「噗」、「哈哈」、「阿 X 同学」、「就喜欢你这么 XX」、字词间加顿号的强调(「我,不,吃」)
- 网文梗:殉职、上道、拿捏、破防
- 自我点破潜台词:「别想多」「开玩笑的」(每幕至多一次,且必须是溜出真话后的找补)、「说好了啊」
- 一切让角色"知道自己在可爱"的台词。幽默必须是干的、冷的、她自己不觉得好笑的。

## 五、真情侣感(最重要的一条,防"合约情侣")

他们**仍然深爱对方**。这个游戏不是两个陌生人在客气,是两个相爱的人在边界前止步。

1. **爱 = 熟稔和记得**,不是甜言蜜语:
   - 她知道他加班就忘吃饭(「我在你楼下。你肯定又没吃饭。」——命令式的关心,老夫老妻的口气)
   - 他知道她不吃香菜、爱溏心蛋;她知道他不吃辣
   - 共享的旧梗、直接的吐槽、自然的肢体距离——**熟,而不是甜**
2. **客气只出现在越界的瞬间**:AA、"不用送"、"先这样"。日常越熟,边界处的突然客气越刺眼——对比就是戏剧。
3. **不确定的是"资格",不是感情**:他们的内心独白围绕"我有没有资格再多要一点?这算不算越界?我们到那一步了吗?"——道德感/边界感的不确定限制依赖,而不是不爱。
4. **藏爱的证据**:在细节里埋"其实很爱"的实据——她包里存着七个月所有 AA 的小票折的方块;"我不吃香菜"这句话她七个月说过四次,每次都只是把丸子夹走,从没生气。

## 六、写作流程

台词写完后依次自问:
1. 她能不能不说这句话,用一个动作代替?(能 → 改成旁白)
2. 超过 15 字了吗?是事务吗?(不是 → 砍)
3. 她知道自己在可爱吗?(知道 → 重写)
4. 感叹号/语气词/梗?(有 → 删)
5. 最重的那句,是不是用了最平的语调?
